## 🖥️ What NimbusPDF Does

NimbusPDF is a private PDF reader for Windows with an offline-capable AI assistant. It keeps your files on your device and gives you a local way to read, search, and ask questions about PDFs.

Use it if you want:

- A PDF reader that works without cloud sign-in
- Local control over your files and notes
- AI help that can run on your own machine
- Support for local AI tools like Ollama and LM Studio
- Optional OpenAI support when you choose to use it
- A setup that fits airgapped or privacy-first systems

## 🤖 Connect the AI Assistant

NimbusPDF can work with a few different AI options.

### Local AI with Ollama

Use Ollama if you want the AI to run on your own computer.

1. Install Ollama on your machine
2. Start the Ollama service
3. Open NimbusPDF settings
4. Select Ollama as the AI provider
5. Enter the local server address
6. Choose a model that fits your PC

### Local AI with LM Studio

Use LM Studio if you want a local model with a simple desktop setup.

1. Open LM Studio
2. Start the local server
3. Copy the server address
4. Open NimbusPDF settings
5. Select LM Studio
6. Paste the server address and save

### OpenAI Support

NimbusPDF can also work with OpenAI if you choose that route.

1. Open NimbusPDF settings
2. Select OpenAI as the provider
3. Enter your API key
4. Save the settings
5. Load a PDF and test the assistant
